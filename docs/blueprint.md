# **App Name**: Trek Explorer

## Core Features:

- Tour/Trek Listing: Fetch tours/treks from Neup.Tourio API, filter by region, type, difficulty, duration.
- Tour Detail Pages: Display detailed tour info (images, itinerary, prices, dates, inclusions, maps).
- Search and Filtering: Enable search and filtering of tours by various criteria (region, price, duration, difficulty).
- Inquiry Submission: Allow users to submit inquiries for custom trips, integrated with Neup.Tourio.
- Image Optimization Suggestion: Provide the user with some prompts that, if run as a tool, will suggest compression and format changes for the provided image, for improved performance.
- Blog Article Display: Showcase blog articles with travel tips and destination guides.
- Team Showcase: Display information about the Happy Mountain Nepal team.

## Style Guidelines:

- Primary color: Earthy green (#4CAF50) to reflect the natural landscapes.
- Background color: Light beige (#F5F5DC) for a clean and neutral base.
- Accent color: Sunrise orange (#FF9800) for call-to-action buttons.
- Body font: 'PT Sans', a humanist sans-serif, will be used for body text to ensure readability.
- Headline font: 'Playfair', a modern serif font similar to Didot, adds elegance to the headlines. Pairing 'Playfair' with 'PT Sans'.
- Use outlined icons to represent different tour activities and amenities.
- Employ a grid layout for tour listings, with card-based previews.
- Implement subtle fade-in animations on image load for a smooth user experience.
